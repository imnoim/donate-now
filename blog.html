What is the Document Object Model (DOM)?

The Document Object Model (DOM) is like a tree of objects for a web page. It lets you use programming languages like JavaScript to change the structure, style, and content of the page. Think of it as a way to make web pages interactive and dynamic

How do you select an element from the DOM?

To select an element from the DOM, use document.getElementById('id') for an element with a specific ID, or document.querySelector('selector') for the first element that matches a CSS selector. These methods help you find and work with elements on a webpage.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is when you add one event listener to a parent element instead of many to child elements. This is useful because it saves memory and makes your code cleaner and faster. For example, one click event on a list can handle clicks for all list items.
